what is function :
		a function is a predefined block of code that can be reused again and again . it helps to reduce repeatation in code and improve code .

console.log(2+4);
console.log(3+4);
console.log(7+5);

this can be reduced to :

function add(a,b);{
	console.log(a+b);
}

add(2+4);
add(3+4);
add(7+5);
